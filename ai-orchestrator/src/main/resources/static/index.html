接口说明

GET /orchestrator/stt/offline?wavUrl= 接收一个可访问的 WAV 文件地址，返回识别出的文本。
示例请求：
curl -G "${location.origin}/orchestrator/stt/offline" --data-urlencode "wavUrl=https://example.com/audio.wav"

如需查看应用运行状况，可访问 /actuator/health。
